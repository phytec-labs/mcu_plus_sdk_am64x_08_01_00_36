Open ETHPHY driver.

Global variables ETHPHY_Config gETHPHY_Config[] and uint32_t gETHPHY_ConfigNum is instantiated by SysCfg to describe the ETHPHY configuration based on user selection in SysCfg.

Parameters

params[IN] Open parameters
Returns
