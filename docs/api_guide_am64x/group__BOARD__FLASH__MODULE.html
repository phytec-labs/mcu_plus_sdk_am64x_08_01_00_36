◆ Flash_open()

Flash_Handle Flash_open ( uint32_t  instanceId,
Flash_Params *  params 
)

Open flash driver.

Make sure the SOC peripheral driver is opened before calling this API. Drivers_open function generated by SysCfg opens the underlying SOC peripheral driver, e.g OSPI.

Internally this API also reads the device and manufacture ID and checks if it matches the expected value for the flash device, if there is mismatch then NULL is returned.

Global variables Flash_Config gFlashConfig[] and uint32_t gFlashConfigNum is instantiated by SysCfg to describe the flash configuration based on user selection in SysCfg.

Parameters
instanceId[in] Index within Flash_Config gFlashConfig[] denoting the flash driver to open
params[in] Open parameters
Returns
Handle to flash driver which should be used in subsequent API call
NULL in case of failure
