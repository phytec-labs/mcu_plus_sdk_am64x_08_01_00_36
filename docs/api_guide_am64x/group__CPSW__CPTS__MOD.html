CPSW CPTS
APIs for Enet LLD » Enet Time Synchronization

Introduction

The CPSW CPTS module provides additional IOCTL commands than those supported by the generic Enet Time Synchronization API set.

CPSW CPTS clocks:

Interrupts:

  • CPSW_EVNT_PEND - Event pending interrupt.

SoC parameters:

  • CpswCpts_Obj::hwPushCnt - Maximum hardware push instances in hardware.

Errata:

  • CPSW_CPTS_ERRATA_GENFN_RECONFIG - GENFn/ESTFn reconfiguration bug.

Compile-time configuration:

  • ENET_CFG_CPSW_CPTS_EVENTS_POOL_SIZE - Maximum size of CPTS event pool.
  • ENET_CFG_CPSW_CPTS_STATS - Event counter (software) statistics. CPTS statistics can be printed via ENET_TIMESYNC_IOCTL_PRINT_STATS.
