CPSW Management Data Input/Output (MDIO)
APIs for Enet LLD » Enet Management Data I/O (MDIO)

Introduction

The MDIO module implements the MDIO-generic IOCTL commands defined in ENET_MOD_MDIO API set.

MDIO clocks:

  • CPSW_CPPI_CLK - CPSW main clock.

Interrupts:

  • MDIO_PEND - MDIO interrupt. Peripheral driver must call MDIO_IOCTL_HANDLE_INTR command to handle the MDIO interrupt.

Features:

  • MDIO_FEATURE_CLAUSE45 - Clause-45 frame feature.

Compile-time configuration:

  • ENET_CFG_MDIO_CLAUSE45 - Clause-45 frame support.
